📄️ 标题（Topic）

随着 CLI 的发展，将命令嵌套在标题中会很有用。只需将命令文件放置在子目录中即可支持此功能。例如，在 Salesforce CLI 中，我们有一个主题 sf config，其中包含类似 sf config set 和 sf config get 的命令。目录结构如下所示：

📄️ 标题分隔符

默认情况下，主题将用冒号分隔，例如 mycommand。但是，如果你喜欢，你可以选择使用空格，例如 my awesome command。

📄️ 钩子

oclif 公开了生命周期事件钩子，例如 init 和 commandnotfound。有关所有生命周期事件的列表，请参见下文。除了这些内置事件外，你还可以创建自己的事件，并允许命令/插件监视这些自定义事件。这是允许多个插件相互交互的好方法。

📄️ 插件

插件是提供实验功能的好方法，允许用户扩展 CLI，将 CLI 分解为模块化组件，或在 CLI 之间共享功能。

📄️ 帮助类

开箱即用的 oclif 为 CLI 提供了很好的帮助体验。用户可以使用 --help 标志调用帮助。

📄️ 性能

oclif 支持开箱即用的性能跟踪 —— 对 oclif 和你自己的代码都是如此。你可以通过两种方式之一来启用它，这取决于你如何实现 bin 脚本。
